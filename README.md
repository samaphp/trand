#Trand
This simple PHP application demonstrates a small retail website.

##Features
- Percentage based discount.
- Dollar value discount.
- Conditional discounts (special for a group of users).
- Affiliate program.
- Simple loyalty program.

##Requirements
- PHP 5.6 or higher.
